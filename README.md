# datauri.py

A simple zero dependency python3 script to convert files (or content) into Data URIs (https://en.wikipedia.org/wiki/Data_URI_scheme).

## Get Started

```bash
$ pip install git+https://github.com/thomaswilley/datauri.git
$ datauri.py -h
```
